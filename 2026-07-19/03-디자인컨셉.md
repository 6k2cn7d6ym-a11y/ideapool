## 피카소 ↔ 달리 비평

### [4] 예약관리

| | A안 (피카소) | B안 (달리) |
|---|---|---|
| 강점 | 진파랑 — 직관적·안전. 비기술자 업주 접근성 우수 | 타임라인 뷰 — 업종 특성(시간 블록) 반영, 바텀시트로 전환 없음 |
| 약점 | 달력 기반은 유사 앱(네이버 캘린더) 대비 차별점 약함 | 테라코타 톤이 업주 연령대(40~50대)에 낯설 수 있음 |

**달리 → A안 비평:** 진파랑+화이트는 무난하지만 비즈니스 앱과 구별이 없다. 업주 입장에서 "내 가게 브랜딩"이 느껴지지 않는다.  
**피카소 → B안 비평:** 타임라인 뷰는 UI 구현 복잡도가 올라간다(예약 블록 겹침 처리). 테라코타는 미용·뷰티 업종엔 맞지만 치과·필라테스엔 이질감.

### [19] 회비관리

| | A안 (피카소) | B안 (달리) |
|---|---|---|
| 강점 | CTA 버튼이 심리 중심 — 클레버 수정사항 가장 직접 반영 | 좌우 분할 + 발송 기록 = 총무 면책 근거 확보 |
| 약점 | "발랄형" 톤이 회비(돈)라는 민감한 주제와 온도차 있음 | CTA가 우측 패널에 묻히면 A안보다 임팩트 약해짐 |

**달리 → A안 비평:** 토스/당근 무드는 좋지만, 회비는 커뮤니티 내 갈등 요소. "발랄"보다 "공식적"이 총무에게 더 실용적이다.  
**피카소 → B안 비평:** 발송 기록 아이디어는 신선하나, 좌우 분할은 모바일 1열 화면에서 레이아웃이 무너진다.

| | A안 (피카소) | B안 (달리) |
