# 아이디어샘물 — 프로젝트 가이드

아이디어 탐색 및 프로토타이핑 공간. 개발팀 사원(팸)의 실험 공간이 포함되어 있으며, 새 라이브러리·프레임워크·API를 검증하는 용도로 사용한다.

> 프로젝트 성격·목적은 아직 정의되지 않음 — 대표님께 확인 필요.

---

## 기술 스택

(확인 필요) — 아직 소스 파일 없음.

---

## 폴더 구조

```
아이디어샘물/
├── .gitignore          — sandbox/ 커밋 제외
└── sandbox/            — 팸(개발팀 사원) 실험 공간
    └── .sandbox-marker — 실험 공간 메타 설명
```

**현재 상태:** 2026-07-17 생성된 빈 폴더. 아직 소스 코드·패키지 없음.

---

## 개발 환경 설정

(확인 필요) — 아직 package.json, 설정 파일 없음.

---

## sandbox 규칙

- `sandbox/` 안 설치는 결재 불요.
- 프로덕션 승격 시 팀장이 의존성 결재 상신.
- `sandbox/`는 `.gitignore`로 커밋 안 됨.

---

## 현재 상태 / 알려진 이슈

- git 미초기화 상태.
- 소스 파일 없음 — 프로젝트 방향 정의 필요.

---

## AI 직원이 작업할 때 주의사항

1. **프로젝트 목적이 아직 미정.** 작업 지시 전에 대표님이 방향을 확정해야 한다.
2. **sandbox/ 안은 자유롭게** 실험 가능. 단 프로덕션 코드로 승격할 때는 팀장 결재 필수.
3. **git 없음.** 초기화 전에 대표님 확인.
4. **배포 방법 미정.** (확인 필요)
